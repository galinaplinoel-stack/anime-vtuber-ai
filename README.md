# 🌸 Anime VTuber AI

An interactive anime VTuber AI companion that you can chat with! Features a cute CSS-animated character with eye tracking and speaking animations.

## Features

- 🎨 **CSS Anime Character** - Fully rendered with CSS art
- 👀 **Eye Tracking** - Character follows your mouse cursor
- 💬 **AI Chat** - Powered by MiMo AI
- ⚡ **Quick Actions** - Pre-set conversation starters
- 📱 **Responsive** - Works on mobile and desktop

## Tech Stack

- HTML5 / CSS3 / JavaScript
- Vercel Serverless Functions
- MiMo AI API

## Live Demo

🔗 [Anime VTuber AI](https://anime-vtuber.vercel.app)

## Setup

1. Clone the repository
2. Set environment variables in Vercel:
   - `AI_API_URL` - API endpoint
   - `AI_API_KEY` - Your API key
   - `AI_API_MODEL` - Model name (optional)
3. Deploy to Vercel
